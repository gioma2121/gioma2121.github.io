02
Research

Subgroup Discovery with LLMs for Sentiment Analysis

Where do models systematically fail?

GitHub ↗

Sentiment classification models work fine on average. The interesting question is: where do they consistently fail? Not randomly — systematically.

I built a framework that uses LLMs to auto-generate features from unstructured text data, then runs subgroup discovery to find the cohorts where the model underperforms. Think of it as teaching models to audit themselves. The LLM-generated features capture patterns that hand-crafted features miss, which is where it gets interesting.

Python LLMs Scikit-learn NLP Subgroup discovery
03
ML Engineering

Fine-Tuning CNNs, ViTs & LLMs for Medical AI

Interpretability in high-stakes settings.

Fine-tuned ResNet-18 (87.1% accuracy) and ViT-B/16 on DermaMNIST, then interpreted the predictions via GradCAM and Attention Rollout. The interesting result wasn't the accuracy number — it was exposing clinically relevant failure modes on the Melanoma/Nevi pair where the model was confidently wrong.

Then went further: applied LoRA via Optuna HPO to fine-tune Gemma-3-270M-IT on MedMCQA, gaining 8.6pp over the random baseline. Built a LlamaIndex RAG pipeline on top that eliminated hallucinations on knowledge-intensive medical queries. The kind of result that makes you trust a system more, not just report better numbers.

PyTorch ViT ResNet LoRA Optuna LlamaIndex GradCAM Gemma 3
04
BSc Thesis

Macrofinancial Effects of US Political Cycles

40 years of data. Deep learning. Political cycles.

Can deep learning surface how US political cycles shape macrofinancial dynamics? That was the question. I built and compared Temporal Fusion Transformer and LSTM models on a 40-year daily dataset combining economic indicators, financial time series, and sentiment data.

TFT outperformed LSTM by around 6x — not because LSTMs are bad, but because TFT's attention mechanism is actually suited to the multi-horizon forecasting problem. Then I interpreted the results via TFT native attention weights and LIME to surface regime-dependent patterns across political cycles. That interpretability layer is what made the thesis worth reading.

PyTorch TFT LSTM LIME Time series Python R
